Create a website hosted for free by GitHub Pages

  1. $ mv index.html to the folder named after your repository
    (Remember repo should be username.github.io)

  2. $ git init in the same "github username" directory

  3. $ git add . files to the stage

  4. $ git commit -m "first commit"

  5. Connect local repo to remote repo
    $ git remote add origin https://username@github.com/username/username.github.io
    Where username is your own github username
    6. and if that doesn't work
  6. $ git remote set-url origin https://username@github.com/username/username.github.io

  7. $ git push --set-upstream origin master
