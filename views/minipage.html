Create a website hosted for free by GitHub Pages

  1. $ mv index.html # ...
    to the folder named after your repository. (Remember repo should be username.github.io)
  2. Git init in the github username directory
  3. Git add . files to stage
  4. Git commit -m "first commit"
  5. Connect local repo to remote repo
  6. git remote add origin https://username@github.com/username/username.github.io

    7. and if that doesn't work

  7. git remote set-url origin https://username@github.com/username/username.github.io
  8. git push --set-upstream origin master
